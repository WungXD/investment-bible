# 日内建仓、持仓、清仓策略（最终版 v5.0）

## 全局约定

- **基准指数**：沪深300（000300），全文"大盘"均指此指数
- **ATR周期**：20日
- **ATR算法**：真实波幅（TR）= max（今日振幅，|今日最高-昨收|，|今日最低-昨收|），ATR = TR的20日简单移动平均
- **ATR取值**：每日收盘后更新，用于次日的止损/止盈判断
- **行业分类**：以申万二级行业为准
- **操作时间基准**：所有时间均为北京时间

---

## 一、建仓

### 1.1 市场状态判断（每日盘前5分钟）

| 市场状态 | 判断条件 | 建仓时间规则 |
|---------|---------|-------------|
| 趋势市 | 沪深300在200日均线之上 + 近5日均成交额 > 20日均成交额 | 试探仓可启用上午窗口 |
| 震荡市 | 沪深300在200日均线之上 + 近5日均成交额 < 20日均成交额 | 全部下午操作 |
| 弱势市 | 沪深300在200日均线之下 | 全部尾盘操作，不加主仓/确认仓 |

### 1.2 仓位分配与间隔

| 批次 | 比例 | 条件（OR关系） | 与上一笔最小间隔 |
|------|------|--------------|----------------|
| 试探仓 | 3-5% | 逻辑确立后首个交易日，按市况选择时间窗口 | — |
| 主仓(1) | 3-4% | 首次回调至5日或10日均线；或缩量企稳（14:50判断） | ≥2个交易日 |
| 主仓(2) | 2-3% | 再次回落到10日或20日均线；或横盘≥5日未创新低 | ≥3个交易日 |
| 确认仓 | 3% | 季报/月度数据验证逻辑后2个交易日内加完 | ≥2个交易日 |

总建仓周期最短7个交易日。三天满仓严格禁止。

### 1.3 试探仓比例确定

| ATR/股价 | 波动等级 | 试探仓比例 |
|----------|---------|-----------|
| > 4% | 高波动 | 3% |
| 2-4% | 中等波动 | 4% |
| < 2% | 低波动 | 5% |

无ATR数据时：近1个月单日最大跌幅>7%→3%，4-7%→4%，<4%→5%。

### 1.4 日内建仓时间窗口

| 建仓批次 | 市场状态 | 时间窗口 | 附加条件 |
|---------|---------|---------|---------|
| 试探仓 | 趋势市 | 10:30-11:30 | 股价回踩15分钟MA20或日内均价线，且未破前低 |
| 试探仓 | 跳空低开（个股无利空） | 10:00-10:30 | 低开1-3%，开盘后10分钟内未继续下探 |
| 试探仓 | 震荡市 | 14:00-14:50 | 默认使用 |
| 主仓(1)/(2) | 所有市况 | 14:00-14:50 | 14:50后判断缩量 |
| 确认仓 | 所有市况 | 14:00-14:50 | 不受高开限制，但涨停买不到则放弃 |
| 全部操作 | 弱势市 | 14:30-14:50 | 指数<200日线，仅试探仓 |

**建仓禁区**（全天任何时段均不建仓）：
- 日内涨幅>4%的个股
- 开盘前30分钟（跳空捡漏除外）
- 放量大阳线当日
- 14:50以后（避免流动性不足滑点）

### 1.5 高开处理规则

| 逻辑来源 | 规则 |
|---------|------|
| 盘后公告（财报/政策） | 高开3-5%：当日可选时间窗口建。高开>5%：当日不买，等次日 |
| 盘中突发 | 当日不追，次日下午再说 |
| 周末发酵 | 周一高开>3%即观望，不开新仓 |

**累计踏空终止条件**：从逻辑确立日算起，累计涨幅>10%且期间无单日跌幅>2%的回调 → 第4个交易日可选时间窗口直接建试探仓，不再等。

### 1.6 主仓B计划（踏空应对）

若10个交易日内没有等来回调（价格始终在5日线之上），第11个交易日14:00后**忽略均线条件**，直接市价加主仓(1)。主仓(2)放弃。

### 1.7 确认仓特殊规则

- 不受高开限制（逻辑已验证）
- 若2个交易日均一字涨停无法买入：放弃确认仓

### 1.8 建仓期风控

- 分批期间累计亏损达总账户1.5%：暂停加仓，重新审视逻辑
- 大盘单日跌超3%：暂停所有加仓，试探仓维持原止损
- 大盘累计跌超5%：不再加仓该标的，试探仓按原止损执行
- 单日不在同一板块买入超过一只标的

### 1.9 跨周末/节假日间隔调整

建仓间隔若跨周末或≥3天的节假日：最小间隔+1个交易日。例：周五建仓 → 下一笔不早于下周三。

## 二、持仓

### 2.1 每日跟踪（约20-25分钟）

| 时间 | 事项 |
|------|------|
| 盘前（5min） | 扫公司公告+行业新闻，判断有无基本面变化；确认当日市场状态 |
| 盘中 | 不操作，除非触发止损/止盈条件单 |
| 收盘后（15min） | ①记录相对强弱：个股涨幅-行业指数涨幅 ②记录成交量vs20日均量 ③检查持仓间相关性 |
| 每周五（30min） | 复查行业数据和主要竞争对手股价走势 |

**“无操作”规则**：如果今日无任何规则触发，则今日操作=不做任何操作。这是正确执行，不是不作为。

### 2.2 逻辑监控三问

每天自问：
1. 我买入时依据的那个核心逻辑，有新的反面证据吗？
2. 行业里有没有其他公司先出问题？
3. 大盘环境变了吗？

### 2.3 动态仓位调整

| 环境 | 单票上限 | 总仓位上限 |
|------|---------|-----------|
| 趋势市（指数>200日线+放量） | 15% | 70% |
| 震荡市（指数>200日线但缩量） | 12% | 50% |
| 弱势市（指数<200日线） | 10% | 30% |

### 2.4 行业集中度

- 同行业持仓 ≤ 2只
- 同行业合计仓位 ≤ 总仓位的25%
- 同行业 = 同一申万二级行业分类

**手动归入同一风险组**：锂矿+锂电池、硅料+硅片等投入品价格高度相关的上下游。

### 2.5 移动止盈（ATR阶梯表）

| 盈利（ATR倍数） | 止损位 |
|----------------|--------|
| < 4 | 成本价 - 2ATR |
| 4 ~ 5 | 成本价 |
| 5 ~ 6 | 成本价 + 1.5ATR |
| 6 ~ 8 | 当前价 - 3ATR |
| 8 ~ 10 | 当前价 - 4ATR |
| > 10 | 当前价 - 5ATR |

阶梯跳变处（如5.9倍→6.0倍）按较宽松执行。

### 2.6 相对强弱行动阈值

每日记录：个股当日涨跌幅% - 行业指数当日涨跌幅%

- **触发关注**：个股5日累计涨跌幅 - 行业5日累计涨跌幅 < -5% → 主动搜索公告，不自动减仓
- **触发减仓**：个股10日累计涨跌幅 - 行业10日累计涨跌幅 < -8% → 减仓50%

### 2.7 持仓横盘处理

持仓超过预期时长的50%后，若股价仍在成本价±1.5倍ATR内：
1. 硬止损收紧至成本价 - 1ATR
2. 再加10个交易日观察期。10日后仍无突破（单日涨幅>3%或创近20日新高）→ 清仓
3. 10日内突破 → 恢复正常ATR止损

横盘期间行业指数已涨超10%而个股不动：减仓70%，不等时间止损。

### 2.8 除权除息调整

- 除权前一日记录所有止损/止盈价位的绝对值
- 除权日按除权比例折算新价位
- 分红到账不影响成本价计算

### 2.9 每周行业复查清单（周五收盘后）

- □ 行业指数 vs 20日均线：跌破 → 该行业所有持仓减至半仓，不加仓
- □ 行业前3大权重股走势：若2/3跌破60日均线 → 行业转弱信号
- □ 该行业是否有公司突然暴跌>10%（排除个股利空）→ 排查是否行业问题
- □ 行业月度数据（如有更新）：和上月比方向对不对
- □ 行业ETF本周资金净流入/流出（如有）

以上5条中≥3条负面：行业逻辑重新评估，该行业仓位不再增加。

## 三、清仓

### 3.1 统一时间规则

| 场景 | 执行时间 | 说明 |
|------|---------|------|
| 紧急止损（基本面暴雷/跳空跌破硬止损） | 次日上午10:00-10:30 | 上午流动性充足 |
| 价格止损（自然跌到止损位） | 次日下午14:00后 | 避开盘中假跌破 |
| 主动止盈（达到目标/时间到期） | 下午14:00-14:30 | 与建仓逻辑一致 |

### 3.2 价格止损细则

- **硬止损**：亏损达2倍ATR，无条件离场
- **时间止损**：逻辑窗口到期，三个交易日内清仓。优先级高于ATR移动止盈。

### 3.3 主动止盈细则

- 分两笔：到位当天下午出60%，剩余40%设ATR移动止盈
- 唯一止盈条件（只赌财报）：全部清仓，不分两笔
- 复合逻辑：仅部分兑现时，对应比例清仓

### 3.4 跳空跌停处理

1. 9:15-9:20集合竞价挂跌停价卖出
2. 若未成交：维持挂单至收盘
3. 连续2个交易日未成交：第3天9:15-9:19撤单 → 9:19-9:25重新挂跌停价
4. 连续3个交易日未成交且封单仍大：不再每日挂单，等复牌或撬板

### 3.5 连续涨停处理

- 封单量>流通盘5%：继续持有
- 连续3个涨停后：每天14:50检查封单量，<1%则次日14:00后清仓
- 第5个涨停起：不论封单，每天14:00后减仓20%

### 3.6 跳空向上处理

| 跳空幅度 | 处理 |
|---------|------|
| 3-5%，有实质利好 | 持有，ATR止盈从跳空前计算 |
| 5-10%，情绪驱动 | 止盈位移到跳空开盘价 |
| >10%，情绪驱动 | 14:00后减仓30%，剩余70%ATR基准价=跳空前一日收盘价 |
| >10%，基本面质变 | 重新评估目标价，不自动减仓 |

判断标准：该利好若发生在买入前，是否会让你重估估值？会→质变；否→情绪。

### 3.7 强行平仓条件

- 公司被立案调查/实控人被采取强制措施
- 审计出具非标意见
- 行业出台颠覆性监管政策
- 大盘暴跌超10%且个股跟跌超8%

### 3.8 止损后重新入场

条件（全部满足）：
1. 距止损出场≥5个交易日
2. 逻辑未被证伪
3. 股价重新站稳20日均线
4. 当前价 ≤ 原止损出场价 × 1.03

规则：新开仓，仓位上限减半。同一标的被止损2次后，该逻辑周期内不再碰。

### 3.9 多标的同日触发止损

- 个股原因导致：一周内不开新仓
- 大盘暴跌导致（沪深300同日跌>3%）：大盘企稳后（连续2日收阳）即可开新仓
- 3只同日触发且同行业：先评估行业逻辑，确认行业问题则全部清完

### 3.10 流动性不足标的（日均成交额<1亿）

- 单票上限8%，单笔不超过2%
- 清仓分散到2-3个交易日

### 3.11 停牌复牌

- 停牌<5日：原策略不变
- 停牌≥5日：复牌首日按开盘价重新判断，开盘破止损则集合竞价出
- 停牌期间行业或大盘重大变化：复牌后重新评估，默认减半仓

## 四、账户与执行

### 4.1 同时持仓数量上限

| 账户资金 | 持仓上限 |
|---------|---------|
| < 50万 | ≤ 3只 |
| 50-200万 | ≤ 5只 |
| > 200万 | ≤ 5只 |

### 4.2 ATR简化替代方案（无ATR数据时）

**近似计算**：ATR ≈ 近20日（最高-最低）均值的70%，或布林带(20)宽度/4。

**完全无ATR时的退化方案**：

| 近1月最大单日跌幅 | 止损距离 | 单票上限 |
|------------------|---------|---------|
| > 8% | -10% | 8% |
| 5-8% | -8% | 12% |
| 3-5% | -6% | 15% |
| < 3% | -5% | 18% |

## 六、操作速查卡

**盘前（5分钟）**：
1. 判断今日市场状态（趋势/震荡/弱势）
2. 确认今日可用建仓时间窗口
3. 检查持仓个股有无突发公告

**盘中**：
- 只看不动，除非条件单触发
- 试探仓上午建仓仅限趋势市回踩或跳空捡漏

**14:00后**：
- 常规建仓/加仓窗口
- 14:50判断缩量最准确

**清仓**：
- 紧急止损：次日上午
- 其余清仓：下午执行
