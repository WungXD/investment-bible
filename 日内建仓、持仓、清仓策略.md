# 日内建仓、持仓、清仓策略（最终版）

## 全局约定

- **基准指数**：沪深300（000300），全文"大盘"均指此指数
- **ATR周期**：20日（非14日），匹配1-3个月持仓周期（算法： 今日振幅、今日最高与昨收差价、今日最低与昨收差价中的最大值，为真实波幅，求真实波幅的20日移动平均）
- **行业分类**：以申万二级行业为准
- **操作时间基准**：所有时间均为北京时间，交易日14:00指下午2点整

---

## 一、建仓

### 1.1 仓位分配

| 批次 | 比例 | 条件（OR关系） | 与上一笔最小间隔 |
|------|------|--------------|----------------|
| 试探仓 | 3-5% | 逻辑确立后首个交易日，14:00后执行 | — |
| 主仓(1) | 3-4% | 首次回调至5日或10日均线；或缩量企稳 | ≥2个交易日 |
| 主仓(2) | 2-3% | 再次回落到10日或20日均线；或横盘≥5日未创新低 | ≥3个交易日 |
| 确认仓 | 3% | 季报/月度数据验证逻辑后2个交易日内加完 | ≥2个交易日 |

总建仓周期最短7个交易日。三天满仓被禁止。

### 1.2 试探仓比例确定

| ATR/股价 | 波动等级 | 试探仓比例 |
|----------|---------|-----------|
| > 4% | 高波动 | 3% |
| 2-4% | 中等波动 | 4% |
| < 2% | 低波动 | 5% |

无ATR数据时：近1个月单日最大跌幅>7%→3%，4-7%→4%，<4%→5%。

### 1.3 高开处理规则

| 逻辑来源 | 规则 |
|---------|------|
| 盘后公告（财报/政策） | 高开3-5%：当日14:00后建。高开>5%：当日不买，等次日 |
| 盘中突发 | 当日不追，次日下午再说 |
| 周末发酵 | 周一高开>3%即观望，不开新仓 |

**累计踏空终止条件**：从逻辑确立日算起，累计涨幅>10%且期间无单日跌幅>2%的回调 → 第4个交易日14:00直接建试探仓，不再等。

### 1.4 日内买点

- **首选时间**：下午14:00-14:50（14:50后全天量基本确定）
- **加仓日优先选阴线尾盘**：跌幅1-3%、成交量在近20个交易日中排名倒数6名以内
- **不在以下情况建仓**：日内涨幅>4%、开盘前30分钟、放量大阳线当日

**缩量判断**：在14:50后执行，"缩量"=当日成交量在近20个交易日中排名倒数6名以内。

### 1.5 主仓B计划（踏空应对）

若10个交易日内没有等来回调（价格始终在5日线之上），第11个交易日14:00后**忽略均线条件**，直接市价加主仓(1)。主仓(2)放弃。

理由：两周（10个交易日）是信息消化周期，超过两周还不追大概率踏空。

### 1.6 确认仓特殊规则

- 确认仓**不受高开限制**（逻辑已验证，高开在预期之内）
- 若2个交易日均一字涨停无法买入：**放弃确认仓**
- 逻辑验证后的高开和普通情绪高开是两回事，不适用回避规则

### 1.7 建仓期风控

- 分批期间累计亏损达总账户1.5%：暂停加仓，重新审视逻辑
- 大盘（沪深300）单日跌超3%：暂停所有加仓，试探仓维持原有止损规则不变
- 大盘累计跌超5%：不再加仓该标的，试探仓按原止损执行
- 单日不在同一板块买入超过一只标的

### 1.8 跨周末/节假日间隔调整

建仓间隔若跨周末或≥3天的节假日：最小间隔+1个交易日。
例：周五建仓 → 下一笔不早于下周三。

## 二、持仓

### 2.1 每日跟踪（约20-25分钟）

| 时间 | 事项 |
|------|------|
| 盘前（5min） | 扫公司公告+行业新闻，判断有无基本面变化 |
| 盘中 | 不操作，除非触发止损/止盈条件单 |
| 收盘后（15min） | ①记录相对强弱：个股涨幅-行业指数涨幅 ②记录成交量vs20日均量 ③检查持仓间相关性 |
| 每周五（30min） | 复查行业数据和主要竞争对手股价走势 |

### 2.2 逻辑监控三问

每天自问：
1. 我买入时依据的那个核心逻辑，有新的反面证据吗？
2. 行业里有没有其他公司先出问题？（同行暴雷往往是领先信号）
3. 大盘环境变了吗？（这个最后考虑，但需要考虑）

### 2.3 动态仓位调整

| 环境 | 单票上限 | 总仓位上限 |
|------|---------|-----------|
| 沪深300在200日均线以上+成交量放大 | 15% | 70% |
| 沪深300在200日均线以上但缩量 | 12% | 50% |
| 沪深300在200日均线以下 | 10% | 30% |

### 2.4 行业集中度

- 同行业持仓 ≤ 2只
- 同行业合计仓位 ≤ 总仓位的25%
- 同行业 = 同一申万二级行业分类

**手动归入同一风险组的情形**（即使申万二级不同）：
- 锂矿+锂电池（同看锂价）
- 硅料+硅片（同看硅价）
- 其他投入品价格高度相关的上下游

判断标准：A公司的核心成本/收入因子与B公司高度重叠，则归入同组。

### 2.5 移动止盈（ATR阶梯表）

| 盈利（ATR倍数） | 止损位 |
|----------------|--------|
| < 4 | 成本价 - 2ATR |
| 4 ~ 5 | 成本价 |
| 5 ~ 6 | 成本价 + 1.5ATR |
| 6 ~ 8 | 当前价 - 3ATR |
| 8 ~ 10 | 当前价 - 4ATR |
| > 10 | 当前价 - 5ATR |

阶梯跳变处（如5.9倍→6.0倍）按较宽松的执行。

### 2.6 相对强弱行动阈值

**每日记录**：个股当日涨跌幅% - 行业指数当日涨跌幅%

- **触发关注**：个股5日累计涨跌幅 - 行业5日累计涨跌幅 < -5% → 主动搜索公司和行业最新公告，不自动减仓
- **触发减仓**：个股10日累计涨跌幅 - 行业10日累计涨跌幅 < -8% → 减仓50%

### 2.7 持仓横盘处理

持仓超过预期时长的50%（预期3个月→过6周，预期2个月→过4周）：

若股价仍在成本价±1.5倍ATR内：
1. 硬止损收紧至成本价 - 1ATR
2. 再加10个交易日观察期。10日后仍无突破（单日涨幅>3%或创近20日新高）→ 清仓
3. 10日内突破 → 恢复正常ATR止损

若横盘期间行业指数已涨超10%而个股不动：减仓70%，不等时间止损。

### 2.8 除权除息调整

- 除权前一日记录所有止损/止盈价位的绝对值
- 除权日按除权比例折算新价位
- 分红到账不影响成本价计算

### 2.9 每周行业复查清单（周五收盘后）

- □ 行业指数 vs 20日均线：跌破 → 该行业所有持仓减至半仓，不加仓
- □ 行业前3大权重股走势：若2/3跌破60日均线 → 行业转弱信号
- □ 该行业是否有公司突然暴跌>10%（排除个股利空）→ 排查是否行业问题
- □ 行业月度数据（如有更新）：和上月比方向对不对
- □ 行业ETF本周资金净流入/流出（如有对应ETF）

以上5条中≥3条负面：行业逻辑重新评估，该行业仓位不再增加。

## 三、清仓

### 3.1 统一时间规则

| 场景 | 执行时间 | 说明 |
|------|---------|------|
| 紧急止损（基本面暴雷/跳空跌破硬止损） | 次日上午10:00-10:30 | 上午流动性充足，价格发现完成 |
| 价格止损（自然跌到止损位） | 次日下午14:00后 | 避过早盘情绪化抛售，确认不是盘中假跌破 |
| 主动止盈（达到目标/时间到期） | 下午14:00-14:30 | 和建仓逻辑一致，等日内价格发现完成再动手 |

**核心原则**：只在紧急情况下上午卖，其余一律下午操作。

### 3.2 价格止损细则

- **硬止损**：亏损达2倍ATR，无条件离场
- **时间止损**：逻辑窗口到期（财报出来了、政策落地了），不管盈亏，三个交易日内清仓
- **时间止损优先级高于ATR移动止盈**：逻辑窗口到期 → 清理全部仓位

### 3.3 主动止盈细则

- **分两笔**：到位当天下午出60%，剩余40%设ATR移动止盈，直到触发或时间到期
- **唯一止盈条件**（如只赌财报超预期）：财报出来次日，全部清仓，不分两笔
- **复合逻辑**（财报+行业趋势），仅财报兑现：清仓60%，剩余40%继续受ATR跟踪

### 3.4 跳空跌停处理

1. 9:15-9:20集合竞价挂跌停价卖出
2. 若未成交：维持挂单至收盘
3. 连续2个交易日未成交：第3天9:15-9:19撤前日挂单 → 9:19-9:25重新挂跌停价
4. 连续3个交易日未成交且封单仍大：不再每日挂单，关注公司公告，等复牌或撬板

### 3.5 连续涨停处理

- 封单量>流通盘的5%：继续持有
- 连续3个涨停后：第4天起每天14:50检查封单量，封单<流通盘1%时次日14:00后清仓
- 第5个涨停起：不论封单，每天14:00后减仓20%

### 3.6 跳空向上处理

| 跳空幅度 | 处理 |
|---------|------|
| 跳空3-5%，有实质利好 | 持有，ATR止盈从跳空前价格计算不变 |
| 跳空5-10%，情绪驱动 | 止盈位移到跳空开盘价（保护跳空收益） |
| 跳空>10%，情绪驱动 | 14:00后减仓30%，剩余70%的ATR止盈基准价=跳空前一日收盘价 |
| 跳空>10%，基本面质变 | 重新评估目标价，不自动减仓 |

**判断标准**：这个利好如果发生在买入之前，会不会让你重新算估值？会→基本面质变，不会→情绪驱动。

### 3.7 强行平仓条件

出现以下情况，无论盈亏直接清仓：
- 公司被立案调查/实控人被采取强制措施
- 审计出具非标意见
- 行业出台颠覆性监管政策
- 持仓期间大盘暴跌超10%且你的票跟跌超8%

### 3.8 止损后重新入场

**条件**（全部满足）：
1. 距止损出场≥5个交易日
2. 逻辑未被证伪
3. 股价重新站稳20日均线
4. 当前价 ≤ 原止损出场价 × 1.03（不得高于止损价3%以上）

**规则**：
- 算新开仓，重新走完整建仓流程
- 仓位上限减半（原15%→7.5%）
- 同一标的被止损2次后，该逻辑周期内不再碰

### 3.9 多标的同日触发止损

- **个股原因导致**（各自独立触发）：都执行，一周内不开新仓
- **大盘暴跌导致**（沪深300同日跌>3%）：都执行，大盘企稳后（连续2日收阳）即可开新仓，不等一周
- **3只同日触发且同行业**：先停，重新评估是否行业逻辑出问题。是→当天全部清完；否→正常各清各的

### 3.10 流动性不足标的（日均成交额<1亿）

- 单票上限从15%降至8%，单笔不超过2%
- 清仓时分散到2-3个交易日执行
- 只用限价单，不用市价单

### 3.11 停牌复牌

- 停牌<5个交易日：原策略不变，停牌期间大盘涨跌忽略
- 停牌≥5个交易日：复牌首日按开盘价重新判断。开盘穿破止损位→集合竞价出；离止损还有距离→等14:00再决策
- 停牌期间行业或大盘发生重大变化：复牌后重新评估逻辑，默认减半仓观察

## 四、账户层面

### 4.1 同时持仓数量上限

| 账户资金 | 持仓上限 |
|---------|---------|
| < 50万 | ≤ 3只 |
| 50-200万 | ≤ 5只 |
| > 200万 | ≤ 5只 |

数量不随资金线性增加，人的精力是瓶颈。每只每日至少15分钟。

### 4.2 ATR计算

**ATR近似计算**：
- ATR ≈ 近20日（最高-最低）均值的70%
- 或：布林带(20)宽度 / 4

**如果以上都算不出来**，退化为固定百分比+仓位补偿：

| 近1月最大单日跌幅 | 止损距离 | 单票上限 |
|------------------|---------|---------|
| > 8% | -10% | 8% |
| 5-8% | -8% | 12% |
| 3-5% | -6% | 15% |
| < 3% | -5% | 18% |

逻辑：高波动票用更宽的止损+更轻的仓位，风险金额大致相当。

## 六、策略执行检查清单

每次操作前过一遍：

- 现在是上午还是下午？上午只执行紧急止损，其余等下午
- 这笔操作是计划内的还是情绪驱动的？
- 当前大盘环境在哪个区间？仓位上限对应调整了吗？
- 同一行业持仓是否已超限？
- 今天是建仓周期的第几天？是否满足最小间隔？
