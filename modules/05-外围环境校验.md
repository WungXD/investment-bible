# 模块05：外围环境校验

## 否决权规则（任一触发 → 暂缓建仓）

- 美联储正处于激进加息周期（单次加息≥50bp）且尚未看到暂停信号
- 北向资金连续10个交易日净流出
- VIX恐慌指数>30且仍在上升
- 人民币汇率在1个月内贬值超过3%
- 行业面临直接的地缘制裁风险（目标公司在实体清单上，或核心供应商被制裁）

---

## 四个外围维度

### 货币政策与流动性

| 信号 | 判断 |
|---|---|
| 央行降准/降息 | 流动性宽松，估值扩张有支撑，中线逻辑更容易兑现 |
| 央行收紧/去杠杆 | 估值压缩可能抵消盈利增长，仓位打折（不超计划的50%） |
| 美联储加息周期 | 外资流出+全球风险偏好降，北向重仓股承压 |
| 美联储降息周期 | 全球流动性宽松，A股外资流入概率增 |

### 全球市场与资本流动

| 信号 | 判断 |
|---|---|
| 美股持续大跌（标普连跌>5%） | 不在暴跌中接飞刀 |
| 北向资金连续5日净流出 | 等流出放缓再建仓 |
| 人民币快速贬值 | 出口受益但整体承压 |
| VIX>30 | 等VIX回落再动手 |

### 地缘政治

- **可对冲风险：** 地缘冲突→黄金/军工；贸易摩擦→国产替代；汇率贬→出口链
- **不可对冲风险：** 全球流动性危机、极端黑天鹅 → 只能减仓/空仓

### 宏观周期定位

| 阶段 | 策略 |
|---|---|
| 复苏期（PMI↑+通胀低+政策松） | 最佳窗口，仓位打满 |
| 过热期（PMI高+通胀↑+政策开始紧） | 缩短持仓周期 |
| 滞胀期（PMI↓+通胀高+政策两难） | 最差环境，回避 |
| 衰退期（PMI↓+通胀↓+政策开始松） | 可左侧布局，分批建 |

## 外围与拐点的矛盾处理

| 情况 | 仓位 |
|---|---|
| 拐点向上 + 外围友好 | 100%（满仓） |
| 拐点向上 + 外围中性 | 70-80% |
| 拐点向上 + 外围恶化 | 30-50%（轻仓） |
| 拐点向上 + 否决权触发 | 0%（等待） |
| 拐点向下（任何外围） | 不参与 |
