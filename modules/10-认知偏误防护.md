# 模块10：认知偏误防护与执行纪律

## 四大致命偏误

| 偏误 | 表现 | 后果 | 对抗机制 |
|---|---|---|---|
| **确认偏误** | 只找支持自己的信息，忽略负面信号 | 该走没走，小亏变大亏 | 建仓时写下证伪条件，定期主动找反驳证据 |
| **锚定效应** | 盯着成本价，"回本就卖"或"亏这么多不如拿着" | 错过最佳卖点或越亏越多 | 屏蔽成本价，只问"以当前价和信息，我会买吗？" |
| **处置效应** | 赚的急于落袋，亏的死拿不卖 | 截断利润，让亏损奔跑 | 退出只看逻辑是否破坏，不看盈亏 |
| **近因偏误** | 过度反应最近的信息，忽视趋势 | 一次坏财报恐慌卖，一次好消息冒进加 | 回看催化剂日历，单次事件放在趋势中判断 |

## 交易备忘录（建仓时强制填写）

```
1. 买入日期与价格：
2. 核心买入逻辑（一句话）：
3. 我的预期差是什么：
4. 预计验证时间：
5. 如果出现___，说明我错了（证伪条件）：
6. 止损位：___（价格或条件）
7. 目标持有周期：
8. 下一个催化剂：___，时间：___
```

**价值：** 当你想"再拿拿看"时翻出来对照——证伪条件触发了吗？触发了就执行，不管感觉。

---

## "冷却期"规则

- **止损触发时：** 先减半（强制），24小时后决定剩余。不给死扣留空间。
- **想加仓时：** 写下理由，等1个交易日。第二天还觉得该加→加；不确定了→昨天是情绪。
- **想抄底时：** 强制等至少2个交易日的稳定期。"接飞刀"欲望最强时最危险。

---

## 复盘框架（每笔交易结束后）

```
1. 买入逻辑对了吗？（判断质量）
2. 执行偏离规则了吗？（执行质量）
3. 赚/亏是运气还是实力？（好判断+坏结果=运气差；坏判断+好结果=运气好）
4. 下次类似情况怎么调整？
```

**核心原则：** 好判断+坏结果 = 继续执行策略；坏判断+好结果 = 必须警觉。

---

## "亏损预算"思维

- 每笔最大可接受亏损 = 总资产的 **2-3%**
- 反推仓位：止损-15% → 单笔仓位 ≤ 总资产的 13-20%
- 这比"我觉得这只很好所以多买"靠谱得多
