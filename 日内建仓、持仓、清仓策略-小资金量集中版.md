# 日内建仓、持仓、清仓策略（小资金量集中账户优化版）

> 适用场景：小资金量账户，重点跟踪并投资1-2只股票。  
> 核心目标：允许适度集中，但用“单笔亏损预算 + 分批间隔 + 总暴露上限”防止一次判断错误伤到账户净值。

---

## 一、优化原则

原版策略适合更分散的组合，单票上限偏低，放在小资金量账户里会导致单笔仓位过碎、交易成本和执行复杂度偏高。集中账户版做三处调整：

1. **持仓数量从“最多3只”改为“主跟踪1只，最多2只”**。
2. **单票仓位上限提高，但每次买入必须受亏损预算约束**。
3. **放弃过细的小比例分批，改成3笔以内完成建仓**，减少小资金量账户里的碎片化交易。

---

## 二、账户层规则

### 2.1 账户仓位上限

| 市场状态 | 总仓位上限 | 单票上限 | 说明 |
|---------|-----------|---------|------|
| 趋势市 | 70% | 核心票45%，备用票25% | 只在逻辑强、指数在200日线上方且放量时使用 |
| 震荡市 | 50% | 核心票35%，备用票20% | 默认状态，优先等下午窗口 |
| 弱势市 | 25% | 单票20% | 只允许试探仓，不加主仓/确认仓 |

**硬约束**：
- 任意时点保留至少30%现金；弱势市保留至少75%现金。
- 两只股票不能属于同一申万二级行业，也不能属于同一风险链条。
- 若只持有1只票，单票最高45%，不得因为“只看好一只”而突破。

### 2.2 单笔亏损预算

小资金量账户按以下亏损预算执行：

| 情况 | 单票最大可承受亏损 |
|------|------------------|
| 试探仓阶段 | 0.6%账户净值 |
| 主仓阶段 | 1.0%账户净值 |
| 已满计划仓位 | 1.5%账户净值 |
| 连续两笔止损后 | 当月停止新开仓 |

仓位计算公式：

```
可买仓位比例 = 本次允许亏损比例 / 止损距离%
```

例：止损距离为8%，本次允许亏损比例为1.0%账户净值，则可买仓位比例 = 1.0% / 8% = 12.5%。  
如果按仓位上限可买更多，也只能买12.5%；风险预算优先于仓位上限。

## 三、建仓优化

### 3.1 大盘与行业前置过滤

建仓顺序固定为：

```
大盘通过 -> 行业通过 -> 个股逻辑通过 -> 日内买点通过 -> 分批建仓
```

对小资金量集中账户来说，大盘和行业不是参考项，而是建仓开关。但开关不能太绝对（强趋势行业经常会短暂跌破20日线后收回），也不能太主观（"明显弱于"必须量化）。

**层级定位**：
- 大盘决定能不能开仓。
- 行业决定能不能加仓。
- 个股决定买哪一只、在哪里买。

#### 三档判断框架

| 状态 | 大盘条件 | 行业条件 | 操作规则 |
|------|---------|---------|----------|
| 通过 | 沪深300在200日均线上方 | 行业不弱于大盘（行业5日涨跌幅 - 沪深300 5日涨跌幅 >= -3%） | 正常建仓，允许试探仓+主仓+确认仓 |
| 部分通过 | 沪深300在200日线上方，但已跌破20日线；或行业短期弱于大盘（行业5日涨跌幅 - 沪深300 5日涨跌幅 在 -3% 到 -5% 之间） | 行业指数在20日线附近但未确认跌破（见下方说明） | 只允许半额试探仓，不开主仓/确认仓 |
| 不通过 | 沪深300跌破200日均线且连续3日未收回；或沪深300单日跌幅>3%（仅当日） | 行业指数跌破20日线且3日内未收回，或跌破时明显放量（成交量 > 20日均量1.5倍）；或行业5日涨跌幅 - 沪深300 5日涨跌幅 < -5% | 不开主仓/确认仓，已有仓位不加仓 |

#### 关键条件量化说明

**行业跌破20日线的确认**（避免假跌破误判）：
- 行业指数跌破20日线后，若3个交易日内收盘收回，视为假跌破，不影响建仓。
- 若3个交易日未收回，或跌破时成交量 > 20日均量1.5倍，才触发"不通过"。

**恢复条件**：
- 大盘从"不通过"恢复为"通过"：沪深300重新站上200日均线且连续3日收于上方。
- 行业从"不通过"恢复为"通过"：行业指数重新站上20日线且连续2日收于上方，或出现放量阳线突破（成交量 > 20日均量且阳线实体>3%）。
- 沪深300单日跌幅>3%的禁令仅限当日，次日若低开企稳（不再创新低）则恢复正常。

**硬规则**：
- 大盘和行业都"不通过"时，不开任何新仓。
- 大盘"不通过"时，不允许买主仓。
- 行业"不通过"时，不允许买确认仓。
- "部分通过"时，试探仓不得超过原计划试探仓的一半。

### 3.2 建仓批次

| 批次 | 默认比例 | 触发条件 | 与上一笔最小间隔 |
|------|---------|---------|----------------|
| 试探仓 | 10%-15% | 大盘与行业过滤后，逻辑确立并在允许时间窗口买入 | - |
| 主仓 | 10%-20% | 回调至5日/10日均线，或14:50缩量企稳 | >=2个交易日 |
| 确认仓 | 5%-10% | 财报、订单、行业数据验证原逻辑 | >=2个交易日 |

单票计划仓位通常为25%-35%；只有趋势市且逻辑被数据验证后，才允许提高到45%。

### 3.3 按波动率调整

| ATR/股价 | 试探仓 | 单票计划上限 |
|----------|--------|-------------|
| > 4% | 10% | 25% |
| 2%-4% | 12% | 35% |
| < 2% | 15% | 45% |

无ATR数据时，用近1个月最大单日跌幅替代：

| 近1月最大单日跌幅 | 试探仓 | 单票计划上限 |
|------------------|--------|-------------|
| > 8% | 8%-10% | 20%-25% |
| 5%-8% | 10%-12% | 25%-30% |
| 3%-5% | 12%-15% | 30%-35% |
| < 3% | 15% | 35%-45% |

### 3.4 小资金量账户的最小交易单位

- 单笔买入低于账户净值8%时，除非是试探仓，否则不做。
- 单笔加仓低于账户净值5%时，合并到下一次有效加仓信号。
- 每只股票最多3笔完成建仓，避免频繁小额交易。

### 3.5 日内时间窗口

沿用原策略：

| 场景 | 可操作时间 | 规则 |
|------|-----------|------|
| 趋势市试探仓 | 10:30-11:30 | 回踩15分钟MA20或日内均价线，且未破前低 |
| 跳空低开捡漏 | 10:00-10:30 | 低开1%-3%，个股无利空，开盘10分钟不再下探 |
| 常规建仓/加仓 | 14:00-14:50 | 默认窗口，14:50判断缩量更可靠 |
| 弱势市 | 14:30-14:50 | 只买试探仓，不加主仓 |

**额外禁区**：
- 日内涨幅>4%不追。
- 放量长阳当天不追。
- 高开>5%不追，至少等次日。
- 当天已经买入一只股票后，不再买第二只。

## 四、持仓优化

### 4.1 集中持仓每日检查

因为只重点做1-2只票，每天只看三类信号：

1. **逻辑是否变弱**：公告、财报、订单、行业数据是否出现反面证据。
2. **价格是否弱于行业**：个股5日表现是否显著跑输行业。
3. **大盘是否变差**：沪深300是否跌破200日线，或单日跌幅超过3%。

### 4.2 集中账户减仓触发

| 触发条件 | 动作 |
|---------|------|
| 单票浮亏达到账户1.0% | 停止加仓，重新检查逻辑 |
| 单票浮亏达到账户1.5% | 无条件减到试探仓或清仓 |
| 个股10日跑输行业指数8%以上 | 减仓50% |
| 行业从"通过"降为"不通过"且个股同步放量下跌 | 减仓到半仓 |
| 沪深300跌破200日线（3日未收回） | 所有持仓降到弱势市上限 |
| 沪深300连续3日累计跌幅>5% | 无论个股状态，全部持仓降至弱势市上限 |

### 4.3 移动止盈

沿用ATR阶梯，但集中账户增加“先收回初始风险”的动作：

| 盈利状态 | 动作 |
|----------|------|
| 浮盈达到1.5倍初始风险 | 止损提到成本价附近 |
| 浮盈达到3倍初始风险 | 减仓30%，剩余仓位按ATR移动止盈 |
| 浮盈达到5倍初始风险 | 再减仓20%，保留尾仓 |
| 股价加速上涨且日内涨幅>7% | 不追买，只移动止盈 |

## 五、清仓优化

### 5.1 硬止损

以下任一条件触发，按清仓规则执行：

- 跌到预设硬止损价。
- 单票亏损达到总账户1.5%。
- 买入逻辑被证伪。
- 公司出现重大负面事件。
- 大盘暴跌超过5%，且个股同步放量下跌。

### 5.2 清仓时间

| 场景 | 时间 |
|------|------|
| 基本面暴雷、跳空跌破硬止损 | 次日10:00-10:30 |
| 正常价格止损 | 次日14:00后 |
| 主动止盈 | 当日14:00-14:30 |
| 连续跌停 | 集合竞价挂跌停价卖出 |

### 5.3 止损后重新入场

必须同时满足：

1. 距离止损出场至少5个交易日。
2. 原买入逻辑没有被证伪。
3. 股价重新站上20日均线。
4. 当前价格不高于原止损价的103%。

重新入场时，计划仓位上限减半。  
同一标的连续止损2次后，本轮逻辑周期内不再交易。

## 六、小资金量账户执行速查

### 盘前

- 判断市场状态：趋势、震荡、弱势（参考2.1表格）。
- 判断大盘与行业前置过滤状态：通过 / 部分通过 / 不通过（参考3.1三档框架）。
- 检查目标股公告和行业新闻。
- 写下今日是否允许买入、加仓、减仓。

### 盘中

- 没有触发规则就不操作。
- 上午只处理趋势市回踩或跳空低开捡漏。
- 追涨冲动出现时，直接等到14:00后再判断。

### 14:00后

- 判断是否符合建仓/加仓窗口。
- 检查成交量是否缩量企稳。
- 若价格已经远离计划买点，不补救，不追。

### 收盘后

填写四项：

```
标的：______
当前仓位：______%
账户最大可能亏损：______%
今日是否触发规则：是 / 否
如果没有触发规则，今日操作 = 不操作
```

## 七、推荐默认参数

对小资金量账户、重点做1-2只票，建议先使用以下默认值：

| 参数 | 默认值 |
|------|--------|
| 同时持仓数量 | 1只为主，最多2只 |
| 单票常规上限 | 35% |
| 单票极限上限 | 45% |
| 总仓位常规上限 | 50% |
| 总仓位趋势市上限 | 70% |
| 单票最大亏损 | 1.5%账户净值 |
| 单笔最小买入 | 8%账户净值 |
| 单票建仓笔数 | 最多3笔 |

这版策略的关键不是“仓位越集中越好”，而是：看对时敢于集中，看错时亏损必须小到第二天还能冷静继续。
