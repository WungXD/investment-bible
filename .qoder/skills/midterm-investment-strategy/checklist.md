# 中线投资策略：决策检查清单

> **使用说明：** 本清单是中线策略的精简入口，每一步是 YES/NO 判断门。通过所有门 → 执行建仓；任一步 NO → 停止或跳转到对应模块查看详细规则。

---

## 第一关：公司能不能买？（财报分析）

→ 详细规则：`modules/01-财报分析.md`

- [ ] 扣非净利润增速 > 去年同期？环比在加速？
- [ ] 经营现金流/净利润比值稳定或改善？（且不低于同行业）
- [ ] 毛利率环比稳定或回升？销售费用率没有异常跑赢营收？
- [ ] 应收账款、存货、合同负债无异常跳升？（跨表联查通过？）
- [ ] 管理层归因与数据一致？（MD&A对得上？）

**任一项 NO → 放弃该标的，不进入下一步**

---

## 第二关：时间对不对？（行业拐点/景气度）

→ 周期股详细规则：`modules/02-行业拐点判断.md`
→ 成长股详细规则：`modules/07-高景气成长股框架.md`

### 先判断：这是周期股还是高景气成长股？

| 条件 | 周期股 | 高景气成长股 |
|---|---|---|
| 增长驱动 | 涨价/供给收缩 | 渗透率/量增 |
| 渗透率 | 成熟（>40%） | 早期（<20%） |
| 行业有无"出清-见底"叙事 | 有 | 无 |

### 周期股检查：
- [ ] 向上拐点前置信号出现？（政策放松/龙头缩减capex/库存低位/价格止跌）
- [ ] 确认信号出现≥2个？（毛利率+产能利用率回升/月度数据改善/小企业退出）
- [ ] 假拐点过滤通过？（真需求非补库/出清彻底无新增产能）

### 高景气股检查：
- [ ] 进场信号≥3个维度满足？（需求爆发/渗透率加速/供给跟不上/业绩验证/产业链印证）

**任一项 NO → 时机未到，等待**

---

## 第三关：价格划算吗？（估值安全边际）

→ 详细规则：`modules/03-估值安全边际.md`

### 周期股：
- [ ] PE/PB在历史 <80% 分位？
- [ ] PEG < 1.5？（<1.0最佳）
- [ ] 一致预期正在上修且未减速？

### 高景气股：
- [ ] PEG（动态）< 1.5？（<1.0最佳）
- [ ] 远期PE合理？（硬科技<20x / 平台型<25x）
- [ ] 渗透率隐含空间未被透支？

**估值已在高位 → 等回调或放弃**

---

## 第四关：我有预期差吗？（最关键的一步）

→ 详细规则：`modules/04-预期差确认.md`

必须能写出以下三句话，写不出 = 不买：

1. **我的预期差是：** _______________
2. **将在___时间内被___事件/数据验证或证伪**
3. **如果我错了，最大亏损约___%（可控）**

快速检查：
- [ ] 我的信息来源是产业链一线，而非券商二手结论？
- [ ] 股价尚未充分反映这个逻辑？（关注度低/一致预期未上修/机构持仓不拥挤）
- [ ] 预期差处于"发现期"或"扩散期早期"，而非"共识期"？

**无预期差 → 即使前三关都过了，也不买**

---

## 第五关：外围配合吗？（外围环境校验）

→ 详细规则：`modules/05-外围环境校验.md`

### 否决权检查（任一触发 → 暂缓建仓）：
- [ ] 美联储未处于激进加息（单次≥50bp）？
- [ ] 北向资金未连续10日净流出？
- [ ] VIX ≤ 30 或已在回落？
- [ ] 人民币1个月内未贬值>3%？
- [ ] 标的未面临直接地缘制裁？

### 环境评估：
- [ ] 当前宏观阶段是？（复苏=满仓 / 过热=缩短周期 / 滞胀=回避 / 衰退=轻仓左侧）
- [ ] 对应仓位调整系数是？（友好=100% / 中性=70-80% / 恶化=30-50%）

**否决权触发 → 设提醒等解除，不建仓**

---

## 第六关：组合装得下吗？（组合管理）

→ 详细规则：`modules/06-组合管理.md`

- [ ] 加入后同行业仓位 ≤ 40%？
- [ ] 加入后同主题/同驱动因子仓位 ≤ 50%？
- [ ] 加入后总持仓数 ≤ 6只？
- [ ] 单只仓位不超限？（周期≤30% / 高景气≤15%）
- [ ] 最坏情况测试通过？（全部止损总亏损 ≤ 10%）
- [ ] 新标的与现有持仓的关联性可接受？

**组合超限 → 要么放弃，要么先卖出一只再建仓**

---

## 全部通过 → 执行建仓

### 建仓执行规则：
1. **分批建仓**：首次不超过目标仓位的40%
2. **写交易备忘录**（强制）：

```
买入日期与价格：
核心逻辑（一句话）：
预期差：
证伪条件：
止损位：
目标持有周期：
下一个催化剂 + 时间：
```

3. **设好催化剂日历提醒**

## 持仓期：按节奏跟踪

→ 详细规则：`modules/08-持仓期管理.md`

| 频率 | 动作 |
|---|---|
| 每日 | 仅看是否触发止损 + 有无突发 |
| 每周 | 高频行业数据 |
| 每月 | 月度数据 + 外围综合评估 |
| 每季 | 财报完整审视 + 重新审视五问 |

### 重新审视五问（每季度/每个催化剂后必答）：
1. 买入逻辑还成立吗？
2. 有新的支持/反驳证据吗？
3. 预期差还在吗？
4. 如果现在空仓，还会以当前价买吗？
5. 下一个催化剂是什么？

**第2/3/4题答案偏负面 → 减仓或退出**

## 退出：规则触发就走

→ 周期股退出规则：`modules/09-退出纪律.md`
→ 高景气股退出规则：`modules/07-高景气成长股框架.md`（第6节）

### 周期股必须退出：
- 毛利率连续2季下滑 **+** 现金流恶化（两个都触发）
- 买入逻辑被反向证据推翻
- 行业向下拐点确认（三项中两项触发）

### 高景气股必须退出：
- 需求增速连续2季环比下降
- 渗透率进入平台期（>40%警惕）
- 供给大规模释放
- 技术路线被颠覆（立即退出）

### 通用纪律退出：
- 止损线触发 → 先减半，24小时后决定剩余
- PE>70%分位 + 盈利上修减速 → 逐步减仓
- 持仓超期但横盘 → 重新审视后决定

---

## 退出后：复盘

→ 详细规则：`modules/10-认知偏误防护.md`

```
1. 买入逻辑对了吗？（判断质量）
2. 执行偏离规则了吗？（执行质量）
3. 赚/亏是运气还是实力？
4. 下次类似情况怎么调整？
```
