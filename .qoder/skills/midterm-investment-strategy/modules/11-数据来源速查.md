# 模块11：数据来源速查

## 通用数据

| 数据类型 | 公开来源 |
|---|---|
| 行业月度数据 | 国家统计局、行业协会官网、Wind/同花顺 |
| 价格数据 | 生意社、各商品资讯网站 |
| 库存数据 | 行业协会季报、上市公司财报里的库存 |
| 调研热度 | 巨潮资讯网（投资者关系记录）、互动易 |
| 行业报告 | 券商研报（**重点关注行业数据和图表，而非结论和估值**） |
| 增减持/回购 | 交易所公告、巨潮资讯网 |

## 外围环境数据

| 数据 | 频率 | 来源 |
|---|---|---|
| 北向资金 | 每日 | 东方财富/同花顺 |
| 人民币汇率 | 每日 | 外汇交易中心 |
| VIX恐慌指数 | 每日（波动时） | CBOE/财经App |
| PMI（中国+全球） | 每月 | 国家统计局、Wind |
| 美联储议息 | 每季度 | 美联储官网、华尔街见闻 |
| 央行货币政策报告 | 每季度 | 央行官网 |
| 地缘事件 | 实时 | 路透、彭博、新华社 |

## 高频另类数据

| 数据 | 价值 |
|---|---|
| 招聘网站岗位变化 | 扩产前先招人 |
| 物流/货运平台数据 | 出货量实时变化 |
| 特定行业指标 | 能繁母猪存栏（猪）、组件排产率（光伏）、北美设备出货额（半导体） |
| 百度/微信指数 | 消费品终端关注度 |

## 高景气框架专用数据

| 数据类型 | 来源 |
|---|---|
| 渗透率与TAM | 行业白皮书（Gartner、IDC、CCID）、头部券商深度报告市场规模测算、产业链投资者交流纪要出货量、下游客户年报capex指引 |
| 产业链高频验证 | 产业链展会纪要（OFC、Computex等）、SEMI北美设备出货额、海关进出口月度数据（特定品类）、龙头月度营收快报（台股/美股） |

---

**核心理念：** 官方数据滞后1个月+，另类数据能在数据真空期提供领先信号。不需要全用，但要有"领先于财报"的信息获取意识。
