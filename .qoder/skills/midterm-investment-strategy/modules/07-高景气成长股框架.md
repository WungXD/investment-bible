# 模块07：高景气成长股框架

## 启用条件（全部满足才用本框架）

- 行业处于新技术/新需求爆发初期（非传统周期复苏）
- 渗透率 <20%（甜蜜区间）
- 需求增速远高于供给增速，无"产能过剩-出清"叙事
- 龙头收入增速连续3季度>30%，主要由**量驱动**（非涨价）

**关键区分：** 涨价驱动（如锂矿2021）= 周期股用主策略；量增驱动（如CPO）= 用本框架。

---

## 财报侧重点（与主策略不同）

| 维度 | 重点 | 原因 |
|---|---|---|
| 收入质量 | 客户集中度、新客户拓展、合同负债趋势 | 只靠1-2大客户的高增长风险极大 |
| 增速"加速度" | 增速在加速还是减速（二阶导） | 减速=估值杀前兆 |
| 研发费用 | 研发资本化比例是否异常升高 | 资本化突升=变相美化利润 |
| 产能进度 | 在建工程转固、爬坡速度 | 产能跟不上→增长无法兑现 |
| 毛利率含义 | 短期下滑可能正常（爬坡/新客户让步） | 不能机械套用周期股规则 |

---

## 进场信号（≥3个维度满足）

| 维度 | 指标 |
|---|---|
| 需求爆发 | 下游capex大幅上调、订单能见度>6个月 |
| 渗透率加速 | 季度渗透率提升加速，处于S曲线陡峭段 |
| 供给跟不上 | 龙头产能利用率>85%且在扩产，交付周期拉长 |
| 业绩验证 | 龙头连续2季营收增速超一致预期10%+ |
| 产业链印证 | 上游订单爆满，各环节互相印证 |

---

## 估值锚定

| 工具 | 安全边际 |
|---|---|
| PEG（动态） | <1.0好；1.0-1.5可接受；>1.5贵 |
| 远期PE（2年后利润） | 硬科技<20x / 平台型<25x |
| 渗透率隐含空间 | 隐含份额假设<实际可能=有空间 |
| PS（亏损期） | PS/营收增速<0.5 |

---

## 仓位管理

| 规则 | 标准 |
|---|---|
| 单只上限 | 15%（主策略可到30%） |
| 建仓方式 | 分批：首次≤目标的40% |
| 加仓条件 | 下季报继续超预期 / 新增量催化 |
| 止损线 | 从最高点回撤20%触发审视 |

---

## 退出纪律

### 必须退出：
- 需求增速连续2季环比下降（即使绝对增速仍高）
- 渗透率进入平台期（>40%高度警惕）
- 供给大规模释放（集中IPO+扩产密集发布）
- 技术路线被颠覆 → **立即退出**
- 龙头开始"讲新故事"（原有引擎衰减）

### 逐步减仓：
- PEG从<1.0升到>1.5 + 预期下修
- 产业链某环节降价抢份额
- 散户涌入 + 媒体铺天盖地
- 核心管理层/早期机构减持

---

## 深度回调处理

| 情况 | 处理 |
|---|---|
| 回调>20%但产业链数据未恶化 | 不自动卖出。PEG回到<1.0 → 可加仓 |
| 回调>20%且数据走弱 | 执行止损，不犹豫 |
| 回调>30%+基本面强+否决权触发中 | 不加不减，等否决权解除 |
| 反弹但缩量+高频数据不再加速 | 警惕构筑顶部 |

**判断标准：** 产业链基本面有没有变。订单加速+客户扩产=送钱；出货走平+客户推迟=逃命。

---

## 外围环境特殊影响

- **利率上行：** 高估值成长首当其冲。仓位打7折。
- **流动性紧+行业爆发矛盾：** 只持有已盈利且PEG<1.0的龙头。
- **板块过热（公募配置比例新高）：** 警惕任何负面消息的踩踏。

---

## 框架切换规则

| 情况 | 处理 |
|---|---|
| 增速降到<20%但格局清晰 | 切回主策略（PE分位+周期逻辑） |
| 技术路线被颠覆 | 立即退出，无容忍期 |
| 看不清哪个框架 | 用更严格的标准（两框架交集） |
