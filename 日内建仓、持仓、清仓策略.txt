❯ 依据上面的策略在选择标后那么日内我该如何建仓、持仓、清仓，策略是什么

  Thought for 4s (ctrl+o to expand)

● 好，前面解决了"买什么"和"什么时候买"的问题，现在解决"怎么买/怎么卖"——这是决定最终收益的最关键环节。

  ---
  日内建仓、持仓、清仓策略

  一、建仓：分批，不死磕开盘价

  中线波段的建仓周期一般在5-15个交易日，不需要一天买完。

  仓位分配

  假设单票目标仓位 15%，分成 3-4 笔：

  ┌────────┬──────┬───────────────────────────────────────────┐
  │  批次  │ 比例 │                   时机                    │
  ├────────┼──────┼───────────────────────────────────────────┤
  │ 试探仓 │ 3-5% │ 逻辑确立后第一时间建，不挑价位            │
  ├────────┼──────┼───────────────────────────────────────────┤
  │ 主仓   │ 5-7% │ 回调至关键均线（20日/60日）或缩量企稳时加 │
  ├────────┼──────┼───────────────────────────────────────────┤
  │ 确认仓 │ 3%   │ 财报或行业数据验证逻辑后，不计价位加完    │
  └────────┴──────┴───────────────────────────────────────────┘

  日内的具体买点

  不要在开盘前30分钟交易——那是情绪化定价最严重的时段，容易买在日内高点。

  比较实用的入场时机：

  - 下午2:00-2:30：日内方向基本确立，流动性也充沛，不容易被日内波动打乱节奏
  - 缩量阴线尾盘：如果当天跌幅在1-3%且成交量明显萎缩（说明不是资金在出逃，只是情绪），尾盘是更好的买入窗口
  - 放量突破时不做：放量大阳线看起来兴奋，但对中线建仓来说追进去成本偏高，等次日是否有回调

  不要做的事

  - 不要在日内追涨超过3%的个股建仓——中线持有成本优势一开始就丢了
  - 不要挂涨停价买——用限价单，比当前价低0.5-1%挂单
  - 不要在情绪激动时（看到利好消息刷屏）进场——消息已经被价格消化了

  ---
  二、持仓：管住手，跟踪三个信号

  持仓期间最大的敌人是过度关注日间波动——每天盯盘会让人把短期噪音误认为趋势反转。

  每日例行（花15分钟就够了）

  ┌────────┬──────────────────────────────────────────────────────────────────────────────────────────────────┐
  │  时间  │                                              做什么                                              │
  ├────────┼──────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ 盘前   │ 扫一眼隔夜行业新闻和公司公告，判断是噪音还是基本面变化                                           │
  ├────────┼──────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ 盘中   │ 除非触发止损/止盈条件，否则不操作                                                                │
  ├────────┼──────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ 收盘后 │ 记录当日收盘价、成交量、相对于行业的超额收益（涨了是不是因为行业普涨？跌了是不是因为大盘暴跌？） │
  └────────┴──────────────────────────────────────────────────────────────────────────────────────────────────┘

  关键是区分个股问题还是大盘问题。如果大盘跌3%你的票跌2%，其实跑赢了，不应该恐慌卖出。

  持仓红线（触发即走，不等日内时机）

  - 基本面证伪：公司公告业绩大幅低于预期、行业政策突然逆转、核心产品出质量问题——无论盈亏，次日开盘直接走
  - 逻辑破坏：你买入时看好的那个核心逻辑不再成立（比如你买的是"行业补库存"，结果行业数据出来库存反而在增加）

  持仓期间的加减仓

  - 加仓条件：季报或月度数据验证你的逻辑 + 股价尚未大涨（逻辑验证后1-2天内加完确认仓）
  - 减仓条件（非清仓）：股价两周内涨超20%以上，且放量滞涨（连续2-3天放量但价格不涨了）——先减掉1/3锁定利润

  ---
  三、清仓：条件触发制，不犹豫

  1-3个月的波段，最怕的是把波段做成股东——本来是等催化剂，结果催化剂没来或被证伪了还不走，越套越深，最后说"改成长期投资吧
  "。这是亏损最大的来源。

  三种清仓场景

  场景一：主动止盈（逻辑兑现）

  股价涨到你的目标位（通常是基于估值或事件催化）：
  - 分两笔走：到位当天出60%，剩下40%观察1-2天，如果继续涨就分批出完，如果回落也出完
  - 不要幻想"卖在最高点"——那是随机事件，不是策略

  场景二：时间止损（最被低估的止损方式）

  你买入时有时间预期——比如"赌这个季度的财报超预期"。如果财报出来了，没超预期，但股价也没跌——很多人会想"再等等"。

  不对。你买入的理由已经到期了，无论盈亏都应该走。时间止损：逻辑窗口关闭 = 离场条件触发。

  场景三：价格止损

  - 硬止损：亏损达到8-10%，无条件离场。不要在跌10%时想"再跌一点就反弹了"——你的研究可能错了
  - 移动止盈：股价涨超15%后，把止盈位上调到成本价+5%以上，确保这笔交易不会从盈利变亏损

  清仓的日内执行

  - 上午10:00-10:30执行：开盘半小时后流动性充足，价格发现完成，不易卖在最低点
  - 不要挂跌停价卖：那是恐慌操作。除非突发利空一字跌停，正常清仓用限价单，比现价低0.5%以内
  - 一天走完：我倾向于一天出清，不分多天。拖成3-5天慢慢卖，反而容易在中间反复纠结

  ---
  总结：一个完整的交易闭环

  选标的（财报+行业拐点）
    → 建仓期5-15天，分3-4笔，下午买
      → 持仓期每日15分钟，只看逻辑变没变，不看盈亏数字
        → 清仓由条件触发：止盈/止损/时间到
          → 清仓后复盘：当初的判断对了还是错了？错在哪？

  最后一点：把清仓后的复盘写下来，哪怕只写两行。三个月后你回头看，会发现反复犯同样的错——那是最值钱的信息。

  错误

  1. 止盈"分两笔"和清仓"一天走完"自相矛盾

  场景一说"到位当天出60%，剩下40%观察1-2天"，但清仓执行又说"一天出清"。两个指令冲突，实际操作会无所适从。

  2. 移动止盈位设得太紧

  涨15%后止盈提到成本+5%，意味着股价从115回调到105就出局。但一只处于上升趋势的股票，回调5-8%是正常波动。这个设置会导致频
  繁被震下车，吃不到主升浪。应该用ATR（平均真实波幅）动态调整，而非固定百分比。

  3. 买卖时间选择标准不统一

  建仓建议下午2:00-2:30（等日内方向确认），清仓又建议上午10:00-10:30（找流动性）。为什么同一个逻辑不适用于两边？——只有紧
  急止损才需要上午赶紧走，计划内的止盈和建仓一样，应该在下午执行。

  。

  - 不在以下情况建仓：日内涨幅>4%、开盘前30分钟、放量大阳线当日（等次日有没有回调）

  建仓期风控

  - 分批期间如果累计亏损已达总账户的1.5%，暂停加仓，重新审视逻辑是否还成立
  - 单日不在同一板块买入超过一只标的

  │      时间       │                                              事项                                              │
  ├─────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────┤

  逻辑监控（核心工作）

  每天自问三个问题：

  1. 我买入时依据的那个核心逻辑，有新的反面证据吗？
  2. 行业里有没有其他公司先出问题？（同行暴雷往往是领先信号）
  3. 大盘环境变了吗？（这个最后考虑，但需要考虑）

  动态仓位调整

  ┌──────────────────────────────────┬──────────┬────────────┐
  │               环境               │ 单票上限 │ 总仓位上限 │
  ├──────────────────────────────────┼──────────┼────────────┤
  │ 指数在200日均线以上 + 成交量放大 │ 15%      │ 70%        │
  ├──────────────────────────────────┼──────────┼────────────┤
  │ 指数在200日均线以上但缩量        │ 12%      │ 50%        │
  ├──────────────────────────────────┼──────────┼────────────┤
  │ 指数在200日均线以下              │ 10%      │ 30%        │
  └──────────────────────────────────┴──────────┴────────────┘

  行业集中度上限：同一行业/产业链所有持仓合计不超过总仓位的25%。如果三只票本质都在赌"半导体周期回升"，那它们算同一行业。

  移动止盈修正（ATR-based）

  放弃固定百分比，改用ATR（14日平均真实波幅）：

  - 盈利达到3倍ATR后，止盈位移到成本价（保本）
  - 盈利达到5倍ATR后，止盈位移到当前价 - 2倍ATR
  - 此后每上涨1倍ATR，止盈位跟随上移1倍ATR

  例：ATR=2元的股票，成本价50元：
  涨到56元（+3ATR）→ 止损提到50元保本
  涨到60元（+5ATR）→ 止损提到56元（60-2×2）
  涨到62元（+6ATR）→ 止损提到58元（62-2×2）

  为什么用ATR而不是固定百分比：一只年化波动率20%的票和一只40%的票，用同一个8%止损，前者给的空间太宽（容忍了4倍常态波动）
  ，后者太窄（一次正常回调就出局）。ATR让止损适配个股的"自然呼吸节奏"。

  三、清仓

  三种场景统一规则

  ┌───────────────────────────────────────┬─────────────────────────────────────────────────────────────────┐
  │                 场景                  │                            执行方式                             │
  ├───────────────────────────────────────┼─────────────────────────────────────────────────────────────────┤
  │ 紧急止损（基本面暴雷/跳空跌破硬止损） │ 次日上午10:00-10:30竞价出清，不挂跌停单，用对手价或低0.5%限价单 │
  ├───────────────────────────────────────┼─────────────────────────────────────────────────────────────────┤
  │ 价格止损（自然跌到止损位）            │ 次日下午2:00后执行，避过早盘情绪化抛售，确认不是盘中假跌破      │
  ├───────────────────────────────────────┼─────────────────────────────────────────────────────────────────┤
  │ 主动止盈（达到目标/时间到期）         │ 下午2:00-2:30执行，和建仓逻辑一致——等日内价格发现完成再动手     │
  └───────────────────────────────────────┴─────────────────────────────────────────────────────────────────┘

  核心原则：只在紧急情况下上午卖，其余一律下午操作。上午是散户情绪交易的高峰，下午是机构定价的时段。

  价格止损细则

  - 硬止损：亏损达2倍ATR（约等同于大部分股票的6-10%），无条件离场
  - 时间止损：逻辑窗口到期（财报出来了、政策落地了），不管盈亏，三个交易日内清仓
  - 跳空跌停处理：一字跌停封单巨大时，排队挂单（挂前一天收盘价的跌停价），同时不要撤销——这是唯一能做的事，不要幻想能卖在
  高位

  主动止盈细则

  - 修正为两笔清仓（消除原版矛盾）：
    - 到位当天下午出60%
    - 剩余40%设移动止盈（ATR规则），让利润跑，直到触发移动止盈或时间止损
  - 涨停持有，不开板不卖；一旦开板回落超3%，次日下午清仓

  强行平仓的额外条件

  出现以下情况，无论盈亏直接清仓：

  - 公司被立案调查/实控人被采取强制措施
  - 审计出具非标意见
  - 行业出台颠覆性监管政策
  - 持仓期间大盘暴跌超10%且你的票跟跌超8%（系统性风险，先出来观察）

  ---
  四、复盘模板

  每笔交易结束后填写，控制在5行以内：

  标的：______  持仓天数：___  盈亏：___%
  买入逻辑：______
  清仓原因：______
  对的判断：______
  错的判断：______
  下次改进：______

  三个月后按"错的判断"归类统计，你会看到自己的系统性偏差——比如"每次都是财报超预期但股价冲高回落"，说明你买在预期里了，以
  后该提前走。

  ---
  核心改动总结：用ATR替代固定百分比（适配波动率）、统一了买卖时间逻辑（紧急→上午，计划→下午）、增加了市场环境自适应仓位
  、补齐了缺口/涨跌停/踏空等极端情况处理。

  1. 试探仓"回避日内涨超4%"与实际可操作性冲突

  说的是"逻辑确立后首个交易日即可建"，又说"涨超4%不建"。但一个重大利好出来后，股价往往直接高开4%+——你的逻辑确立和避免涨
  超4%不可能同时满足。需要区分：逻辑来源如果是盘后公告（次日高开是大概率），早上进场还是放弃？原方案没给。

  2. 主仓B计划的"第10个交易日尾盘"——10天没有依据

  为什么是10天而不是8天或15天？而且ATR部分用的是严谨的量化逻辑，这里突然拍了一个整数，风格不统一。其实应该和ATR联动：如
  果股价沿5日线稳步上行且一直没回踩ATR支撑区，到什么程度意味着趋势确认该直接加了。

  3. 移动止盈中"3倍ATR后止损提至保本"——保本太早了吗？

  ATR=2元，成本50。涨到56（3ATR），止损提到50。如果这是个波动大的票（日波动1.5-2元很正常），一次日内回调就可能从56跌回50
  触发保本止损，然后股价反身向上。你的逻辑没错但盈利被洗掉了。3倍ATR可能太浅。

  4. 时间止损给了"三个交易日"，主动止盈剩余仓位又在等ATR跟踪

  如果时间是主动止盈的触发条件（比如季报出来了，兑现了），但还剩40%仓位在等ATR移动止盈——这两套规则同时作用于同一笔持仓，
  到底哪个优先级更高？时间到了就该清，不应该还有40%在"等利润跑"。

  三、细节不完善

  1. 建仓间隔时间没说明

  分4次建仓，每次间隔至少多久？一天买两笔算两次吗？如果是试探当天就买、第二天回调就加主仓、第三天接着确认仓——三天买完15%
  ，风险太集中。应该设最小间隔。

  2. "成交量低于20日均量50%"作为缩量加仓条件——这个阈值的问题

  震荡市里成交量本身就在萎缩，20日均量持续下行，用50%的阈值要么总触发（不断加仓）要么永远不触发。应该用成交量分位数而不
  是固定比例。

  3. 停牌复牌没有和止损联动

  停牌期间大盘或行业跌了10%，复牌补跌，你的止损单在复牌集合竞价直接触发，但实际上挂不出去。这种情况应预先说明。

  若10个交易日内没有等来回调（价格始终在5日线之上），视为趋势强于预期。在第11个交易日14:00后加主仓(1)。如果此后继续沿5日

  二、持仓（修订）

  移动止盈修正（ATR倍数调整）

  原版"3倍ATR保本"确实太浅。改用：

  - 盈利达4倍ATR：止盈位移到成本价
  - 盈利达6倍ATR：止盈位移到当前价 - 3倍ATR
  - 此后每上涨2倍ATR：止盈位跟踪上移1倍ATR

  以ATR=2元，成本50元为例：
  涨到58元（+4ATR）→ 止损提到50元
  涨到62元（+6ATR）→ 止损提到56元（62-3×2=56）
  涨到66元（+8ATR）→ 止损提到58元（66-3×2→更新为66-4×2=58，再算：每个2ATR上移1ATR，即涨4元止盈上移2元，56→58）

  说人话就是：刚开始盈利时给更多波动空间，别被洗出去；利润越厚，移动止盈越紧。

  除权除息调整

  持仓期间遇到除权除息：
  - 在除权前一日记录前复权价格
  - 所有止损/止盈价位在除权日开盘前按复权因子等比例调整
  - 分红到账不影响成本价计算（成本价不调）

  时间止损与剩余仓位

  消除矛盾：时间止损优先级高于ATR移动止盈。逻辑窗口到期 → 清理全部仓位，最后那40%不等"让利润跑"。

  三、清仓（修订）

  极端情况处理

  连续涨停
  - 封单量 > 流通盘的5%：继续持有
  - 连续3个涨停后：若第4天仍封死涨停，继续持有但每天14:50检查封单量；封单一旦小于流通盘的1%，次日14:00后清仓
  - 第5个涨停起：不论封单，每天14:00后减仓20%

  停牌复牌
  - 停牌<5个交易日：原策略不变，停牌期间大盘涨跌忽略，复牌后按前一日价格和止损位继续执行
  - 停牌≥5个交易日：复牌首日按开盘价重新判断——如果开盘就穿破止损位，集合竞价出；如果离止损还有距离，等14:00再做决策
  - 停牌期间行业或大盘发生重大变化：复牌后重新评估逻辑是否还成立，默认减半仓观察

  流动性不足的标的（日均成交额<1亿）
  - 单票上限从15%降至8%，单笔不超过2%
  - 清仓时分散到2-3个交易日执行（每个交易日14:00后出一部分）
  - 只用限价单，不用市价单

  - 同日触发：都执行，不分先后

  第三轮审查

  一、新发现的逻辑硬伤

  1. 试探仓"涨超4%等第二天"——没有终止条件

  若当日涨超+4%，第二天14:00再试

  那第二天又涨超4%呢？第五天呢？如果逻辑确立后股价连涨一周，规则会让你无限期等下去，最后在更高的位置追进去。需要一个最大
  等待天数，超了就放弃或追。

  2. 确认仓与高开规则直接冲突

  确认仓规则："季报/月度数据验证逻辑后2个交易日内加完"

  高开规则："高开>5%：当日不买，等次日"

  问题是：季报超预期验证了你的逻辑，第二天股价大概率高开。
  如果高开6%，确认仓是加还是不加？两个规则指向相反方向。应该在确认仓上给豁免——逻辑验证后的高开和普通情绪高开是两回事。

  3. 试探仓建了之后，大盘暴跌——手里已有仓位怎么办？

  说的是"暂停对该标的的一切操作，重新评估"。但没说这3-5%的试探仓是拿着还是平掉。如果继续拿着等"重新评估"，评估期间再跌，
  小仓位变中等亏损。应该明确：试探仓此时跟随原止损规则，还是豁免等评估完。

  4. 移动止盈在+4到+6 ATR之间是空白地带

  价格从58涨到62这个区间（+4到+6 ATR），止损一直停在50（成本价）。但万一股价从60回落到51呢？亏损1元然后反弹到65——你的止
  损在50没触发，但坐了过山车。问题在于：+4 ATR保本之后，止损在相当长一段涨幅里原地不动，没有渐进上移。
  应该线性插值而非跳跃。

  5. 时间止损 vs ATR移动止盈——上一轮说优先级已解决，实际没解决干净

  说的是"时间止损覆盖一切"、"逻辑窗口到期全部清仓"。但复合逻辑场景下：

  ▎ 复合逻辑（财报+行业趋势），仅财报兑现 → 清仓60%，剩余40%继续受ATR跟踪和时间双约束

  "时间双约束"是什么意思？如果行业趋势那个逻辑的验证窗口是两个月后，但ATR在中间就触发了止盈——到底是走还是等？两个约束谁
  先到谁生效？表述含糊。

  6. 相对强弱每日记录但没有任何行动阈值

  持仓期每天算"当日涨幅 - 行业指数涨幅"，但没说什么情况下这个数据会触发操作。如果连续5天跑输行业2%但还没到止损位，是预警
  信号还是忽略？记录不用等于白记。

  ---
  二、遗漏的关键场景

  7. 持仓横盘——最消耗耐心的场景

  1-3个月的波段，最怕的不是暴跌（止损会保护你），而是持仓6周股价原地不动。逻辑没破、没到止损、也没兑现——每天看着别的票涨
  自己的不动。多数人会在第5-8周因为失去耐心而主动卖掉，然后股价起飞。

  原策略没有任何处理横盘的规则。需要加一条："持仓超过预期持有时长的一半（比如预期2个月，过了1个月）而股价仍在成本价±1ATR
  范围内窄幅波动"，触发什么动作。

  8. 止损后股价反弹——能不能接回来

  止损出去了，三天后股价回到原逻辑轨道上。能不能重新买？如果能，算新开仓还是续持仓？建仓规则从头来还是加速？多数人会因为
  "刚亏过"而不敢买，但往往错失最大的机会。

  9. 同时持有几只票——数量上限

  策略一直假设单票操作，但实际账户会同时持有2-5只。每只都需要每日跟踪、每只都有止损止盈要盯。数量超了管不过来。应该建议
  同时持有的上限（和账户规模、个人时间相关）。

  10. 大账户 vs 小账户——单票15%在50万账户和500万账户的区别

  50万账户单票15%=7.5万，买一只50元的票=1500股=7.5万刚好。但如果是20元以下的票，最小交易单位限制可能让仓位偏离目标。影响
  不大但该提一句。

  三、参数和计算的细节问题

  11. ATR的计算周期

  策略全程用ATR但没说明取多少日。14日是常规默认，但日线ATR和你的交易周期（1-3个月）匹配吗？对于波段交易，也许用20日ATR更
  稳定，减少单日异常波动对止损位的干扰。

  12. 主仓(1)缩量条件可操作性差

  ▎ 成交量处于近20日最低30%分位

  普通投资者没有系统自动算分位数。更实用的表述："当日成交量在近20个交易日中排倒数6名以内"——同一件事，但在看盘软件上能直
  接目测。

  13. 高开规则中"大盘"未定义

  多处提到"大盘"，但没说是上证综指、深证成指还是沪深300。不同指数走势可能分化。应该固定一个基准。

  14. "每周五复查行业数据"——查什么、触发什么

  只有一句话没展开。应该列具体检查项和对应的决策：
  - 行业指数是否跌破20日均线？→ 减仓
  - 是否有同行突然暴跌？→ 排查原因
  - 行业月度数据是否出问题了？→ 重新评估逻辑

  四、矛盾和表述问题

  15. 清仓时间逻辑还没统一干净

  ┌──────────────────────┬─────────────────────┬────────────────┐
  │         场景         │      当前规则       │      问题      │
  ├──────────────────────┼─────────────────────┼────────────────┤
  │ 紧急止损             │ 次日上午10:00-10:30 │ 对             │
  ├──────────────────────┼─────────────────────┼────────────────┤
  │ 价格止损             │ 次日下午2:00后      │ 对             │
  ├──────────────────────┼─────────────────────┼────────────────┤
  │ 主动止盈             │ 下午2:00-2:30       │ 对             │
  ├──────────────────────┼─────────────────────┼────────────────┤

  └──────────────────────┴─────────────────────┴────────────────┘

  跳空跌停的挂单时间没说——如果是竞价阶段就要挂，那是9:15之前，和其他所有规则的时间都不一样。

  16. 除权调整用词不准确

  ▎ 在除权前一日记录前复权价格

  "前复权"是向后调整历史价格，除权当日需要的是向前调整止损价位。应改为："除权前一日记录所有止损/止盈价位的绝对值，除权日
  按除权比例折算新价位"。

  修订版（第三轮）

  只写有改动的部分，其余沿用上一版。

  ---
  一、建仓

  1. 试探仓高开回避——加终止条件

  逻辑确立后首个交易日：
  - 若当日涨幅≤4%：14:00后建试探仓
  - 若当日涨幅>4%：次日14:00再试
  - 连续3个交易日涨幅均>4%（累计踏空超10%）：第4个交易日14:00直接建试探仓，不再等

  理由：连涨三天说明市场共识极强，继续等下去成本越等越高。三天是容忍上限。

  2. 确认仓——豁免高开规则

  确认仓不受高开限制。
  季报/月度数据验证逻辑后，2个交易日内14:00后加完。
  即使当日高开>5%，逻辑验证本身是新增信息，高开在预期之内。

  确认仓和普通建仓的区别：普通建仓时你是在"赌"逻辑成立，确认仓是逻辑已经被验证。后者不适用情绪回避规则。

  3. 建仓期大盘暴跌——明确试探仓去留

  建仓期间如果大盘（沪深300）单日跌超3%：
  - 暂停所有加仓
  - 试探仓（已持有部分）维持原有止损规则不变（2倍ATR硬止损）
  - 若大盘累计跌超5%：不再加仓该标的，试探仓按原止损执行，不额外操作

  理由：试探仓仓位小（3-5%），不需要特殊保护；该止损就止损，不该止损就拿着。不加仓是因为环境变了，原逻辑需要在大盘企稳后
  重新确认。

  ---
  二、持仓

  4. 移动止盈——填补+4到+6 ATR空白

  盈利<4倍ATR：止损 = 成本价 - 2倍ATR（硬止损）
  盈利≥4倍ATR且<6倍ATR：止损从成本价线性上移至(当前价-3倍ATR)
    - 即：止损 = 成本价 + (盈利 - 4×ATR) × 1.5
    - 例：ATR=2，价格60（+5ATR），止损 = 50 + (10-8) × 1.5 = 53
  盈利≥6倍ATR：止损 = 当前价 - 3倍ATR
  此后每上涨2倍ATR，止损距当前价再放宽1倍ATR
    - 8倍ATR：止损距当前价4倍ATR
    - 10倍ATR：止损距当前价5倍ATR

  原版从保本直接跳到3倍ATR缓冲有断层。线性插值让止损在+4到+6之间渐进上移，不会让利润全部暴露。

  5. "时间双约束"——明确定义

  复合逻辑（如：财报超预期 + 行业补库回升）：
  - 任一逻辑窗口到期且该逻辑被证伪 → 清仓该逻辑对应的仓位比例
  - 任一逻辑窗口到期且被验证 → 只清该逻辑对应的仓位，其余继续
  - ATR止损和时间止损各自独立运作，谁先触发谁先执行
  - 剩余仓位在剩余逻辑的窗口期内继续受ATR规则保护

  简单说：每一条买入逻辑都独立对应一笔仓位，各自有各自的到期日和止损线。

  6. 相对强弱——加行动阈值

  每日记录：个股涨幅 - 行业指数涨幅（均用收盘价算）

  触发关注：连续5个交易日累计跑输行业超5%
    → 检查是否有利空被市场逐步消化而你忽略了
    → 不自动减仓，但主动搜索公司和行业最新公告

  触发减仓：连续10个交易日累计跑输行业超8%
    → 减仓50%。你的逻辑可能已经反映在行业涨跌里，但个股层面出了问题你没发现

  7. 持仓横盘——新增处理规则

  持仓超过预期时长的50%（预期3个月→过6周，预期2个月→过4周）：

  若股价仍在成本价±1.5倍ATR范围内（相当于没涨没跌）：
  - 检查：逻辑还在不在？催化剂时间是否推迟了？
  - 若逻辑仍在但催化剂推迟：继续持有，但将硬止损收紧至1.5倍ATR（压缩时间成本）
  - 若逻辑模糊不清：减仓50%，剩下等原时间止损到期

  若横盘期间行业指数已涨超10%而个股不动：
  - 个股明显弱势，减仓70%。你的选股可能出了问题

  理由：横盘最杀人，需要规则帮你做决策而不是靠意志力扛。

  三、清仓

  8. 跳空跌停——明确挂单时间

  跳空跌停（开盘即封死）：
  - 9:15集合竞价阶段即挂跌停价卖出
  - 若集合竞价未成交，维持挂单至收盘
  - 若连续2个交易日一字跌停未成交：第3天开盘前撤单，改为9:20挂对手价（有人撬板时优先成交）

  9. 止损后重新入场规则

  止损出场后，若股价重新回到逻辑轨道：

  条件（全部满足才可重新入场）：
  1. 距止损出场至少5个交易日
  2. 逻辑未被证伪（你被止损是因为价格波动而非基本面变化）
  3. 股价重新站稳20日均线

  重新入场规则：
  - 算新开仓，重新走完整建仓流程（3-4笔，至少7天）
  - 仓位上限减半（原15%→7.5%）
  - 同一标的被止损2次后，该逻辑周期内不再碰

  原因：二次入场风险更高——要么你逻辑错了，要么市场不认你的逻辑。
  两种情况下都应该降低暴露。

  10. 跳空向上>10%——区分事件类型

  跳空>10%：
  - 若是基本面质变（重组、重大合同、新品获批等改变公司价值的）：
    → 重新评估目标价，不自动减仓
  - 若是情绪驱动（板块轮动、游资炒作、跟风外围）：
    → 14:00后减仓30%，锁定暴利
  - 若无法判断：默认按情绪处理减仓30%

  判断标准：这个利好如果发生在你买入之前，会不会让你重新算估值？
  会 → 基本面质变，不会 → 情绪驱动。

  四、账户层面

  11. 同时持仓数量上限

  同时持仓标的数 ≤ 5只
  精力预算：每只每日至少15分钟，5只=1.25小时/日，超过管不过来

  若账户资金<50万：≤ 3只（单票最低交易金额有时限制分散度）
  若账户资金50-200万：≤ 5只
  若账户资金>200万：≤ 5只（数量不随资金线性增加，人的精力是瓶颈）

  同行业持仓 ≤ 2只
  同产业链（上下游）算同行业

  12. 每周行业复查——具体清单

  每周五收盘后（30分钟）：

  □ 行业指数 vs 20日均线：跌破 → 该行业所有持仓减至半仓，不加仓
  □ 行业前3大权重股走势：若2/3跌破60日均线 → 行业转弱信号
  □ 该行业是否有公司突然暴跌>10%（排除个股利空）→ 排查是否行业问题
  □ 行业月度数据（如有更新）：和上月比方向对不对
  □ 北向资金本周在该行业是净流入还是净流出

  以上5条中≥3条负面：行业逻辑需要重新评估，该行业仓位不再增加。

  全文"大盘"统一为沪深300（000300）。它覆盖两市、代表主流机构定价，比上证综指更能反映真实市场环境。

  14. ATR参数明确

  值对止损位的扰动。

  一、建仓部分

  问题 1：确认仓豁免高开，但遇到涨停呢？

  确认仓不受高开限制。即使当日高开>5%，逻辑验证本身是新增信息，高开在预期之内。

  季报盘后超预期 → 第二天一字涨停 → 第三天继续一字涨停。你的规则是"2个交易日内加完"，两天都买不到怎么办？

  需要终止条件：2个交易日未成交则放弃确认仓，不追第三个交易日。 逻辑已兑现但市场不给你上车机会，这钱不是你该赚的。

  问题 2：主仓(1)和主仓(2)的条件区分度太低

  ┌─────────┬──────────────────────────────────┐
  │  批次   │               条件               │
  ├─────────┼──────────────────────────────────┤
  │ 主仓(1) │ 回调至5日/10日均线且缩量         │
  ├─────────┼──────────────────────────────────┤
  │ 主仓(2) │ 回踩均线，或横盘≥5日未破20日均线 │
  └─────────┴──────────────────────────────────┘

  "回调至5日线"和"回踩5日线"是同一件事。两笔加的时机几乎重叠。而且如果主仓(1)在回调到5日线时已经加了，主仓(2)再等"回踩均
  线"——股价可能已经反弹离开均线了，根本不会回踩第二次。

  主仓(2)的条件应该和主仓(1)互斥：
  - 主仓(1)：价格首次回调至5日/10日均线且缩量 → 加
  - 主仓(2)：加完主仓(1)后，价格再次回落到10日/20日均线（更深回调）→ 加；或横盘整理≥5日未创新低 → 加

  这样两笔加仓对应不同的市场状态：主仓(1)吃浅回调，主仓(2)吃深回调或时间消化。

  问题 3：连续3日涨幅>4%的定义有歧义

  ▎ 连续3个交易日涨幅均>4%

  如果 Day1 涨5%、Day2 涨3.5%、Day3 涨5%——这算"连续"被中断了吗？Day2没超过4%，但三天累计涨了14%，你还是在踏空。

  应该改为累计条件：从逻辑确立日算起，累计涨幅>10%且期间无单日跌幅>2%的回调机会 → 第4个交易日直接建。

  问题 4：主仓B计划第11天加仓——此时价格可能已经远离均线，和主仓(1)的条件矛盾

  ▎ 第11个交易日14:00后加主仓(1)

  但主仓(1)的条件是"回调至5日/10日均线"。第11天股价可能在5日线上方5%的位置，根本不满足主仓(1)的条件。此时加仓用的是哪条
  规则的授权？没说清楚。

  B计划触发时应明确：忽略均线回调条件，直接以当日14:00市价加主仓(1)。 它是独立于常规主仓(1)的例外规则，不是补丁。

  问题 5：试探仓取3%还是5%没有标准

  同样是试探，一个票只上3%，另一个上5%，决策依据是什么？应该和ATR或波动率挂钩：

  - ATR/股价 > 4%（高波动）：试探仓取3%
  - ATR/股价 2-4%（中等波动）：试探仓取4%
  - ATR/股价 < 2%（低波动）：试探仓取5%

  波动越大的票试探仓越轻，因为止损距离更远（ATR大），同等仓位下实际风险金额不同。

  二、持仓部分

  问题 6：移动止盈的线性插值公式——实战中算不出来

  止损 = 成本价 + (盈利 - 4×ATR) × 1.5

  这是盘中实时决策，不是盘后做作业。没人能在14:25掏出计算器摁这个。需要一张查表或简化版。

  简化为阶梯表：

  ┌─────────────────┬─────────────────┐
  │ 盈利（ATR倍数） │     止损位      │
  ├─────────────────┼─────────────────┤
  │ < 4             │ 成本价 - 2ATR   │
  ├─────────────────┼─────────────────┤
  │ 4 ~ 5           │ 成本价          │
  ├─────────────────┼─────────────────┤
  │ 5 ~ 6           │ 成本价 + 1.5ATR │
  ├─────────────────┼─────────────────┤
  │ 6 ~ 8           │ 当前价 - 3ATR   │
  ├─────────────────┼─────────────────┤
  │ 8 ~ 10          │ 当前价 - 4ATR   │
  ├─────────────────┼─────────────────┤
  │ > 10            │ 当前价 - 5ATR   │
  └─────────────────┴─────────────────┘

  实际使用时看一眼当前盈利是ATR的几倍，对号入座。精确到小数点后一位没有必要——止损本来就不是精密科学。

  问题 7：横盘收紧止损——力度不够

  ▎ 硬止损收紧至1.5倍ATR

  原来硬止损是成本价 - 2ATR，收紧到成本价 -
  1.5ATR。ATR=2元时，止损从46提到47——只差了1元。对于一只横盘6周的股票，1元的止损收紧杯水车薪。

  应该是：硬止损收紧至成本价 -
  1ATR，同时加一条时间硬止损——横盘超过预期时长50%后再给最多10个交易日，仍无突破则清仓。两个条件任一触发即走。

  问题 8：相对强弱用加法还是乘法没说明

  ▎ 连续5个交易日累计跑输行业超5%

  Day1 跑输1.2%，Day2 跑输1.3%，Day3 跑输1.1%，Day4 跑输0.8%，Day5
  跑输0.9%——加在一起是5.3%，触发了。但这5天个股每天只比行业弱1%左右，属于正常波动范围内的偏弱而非异常。

  累计跑输应该用复利累积：个股5日累计涨幅 - 行业5日累计涨幅 <
  -5%。这和每日差额的简单加总不同——前者考虑了基数效应，后者会高估偏离度。

  实际执行中："过去5个交易日，个股累计涨跌幅 - 行业指数累计涨跌幅 < -5%"。 看盘软件上两个数字减一下就行，不用每天加总。

  问题 9：北向资金行业统计——普通投资者做不到

  ▎ 北向资金本周在该行业是净流入还是净流出

  北向资金数据按个股披露，要汇总到行业需要写脚本或手工加总几十只股票。这个检查项对普通投资者不现实。

  改为：查看行业ETF（如有）本周资金净流入/流出——ETF的份额变化和资金流向数据是公开的，在东方财富、天天基金等平台直接可查
  。

  ---
  三、清仓部分

  问题 10：跳空跌停处理——9:20的时间窗口错误

  ▎ 第3天开盘前撤单，改为9:20挂对手价

  A股集合竞价规则：9:15-9:20 可挂可撤，9:20-9:25 只能挂不能撤。9:20这个时间点是撤单截止的最后一秒，过了9:20:00就不能撤了
  。"9:20挂"等于在不可撤单阶段挂单，前面挂的单如果没撤掉就冲突了。

  修正：
  第3天：9:15-9:19之间撤掉前两天的挂单
          9:19-9:25之间重新挂跌停价（或对手价，赌有人撬板）
          如果还是未成交：收盘后考虑通过大宗交易或放弃（流动性已枯竭，不恋战）

  问题 11：重新入场——5天后可能已经追高

  ▎ 距止损出场至少5个交易日 + 股价重新站稳20日均线

  一只票你46止损出场，5天后股价48重新站稳20日均线。你重新入场成本比当初止损时高了2元（4.3%）。而且你是被止损出去的，说明
  之前判断有问题——同样的逻辑、更高的价格重新买，凭什么这次就对？

  加一条前置条件：重新入场价不得高于原止损出场价的3%。 如果高于3%，放弃——你已经被市场证明错了，不追。

  问题 12：跳空向上减仓30%后——剩余仓位的ATR基准从哪算

  ▎ 跳空>10%情绪驱动 → 14:00后减仓30%

  减了30%后，剩余70%的ATR止盈从跳空前价格算还是跳空后价格算？如果是情绪驱动而减仓，跳空这部分"虚涨"随时可能回吐——止盈应
  该从跳空前一日收盘价起算，而不是跳空后的新高。这样保护那30%已锁利润的同时，剩余仓位也有一道防线。

  明确：减仓后剩余仓位的ATR止盈基准价 = 跳空前一日收盘价。

  问题 13：多标的止损后禁止开新仓——可能错过最佳买点

  ▎ 多标的同时触发止损后的一周内，不开新仓

  如果是大盘暴跌引发集体止损，一周后大盘企稳，情绪修复初期往往是最好的买点，这个规则会让你踏空。

  区分两种情况：
  - 个股原因导致的集体止损（各自独立触发）= 一周不开新仓（你的策略可能系统性出问题了）
  - 大盘暴跌导致的集体止损（沪深300单日跌>3%）= 大盘企稳后（连续2日收阳）就可以开新仓，不需要等一周

  ---
  四、账户与执行

  问题 14："同产业链算同行业"太宽

  光伏玻璃（福莱特）和光伏逆变器（阳光电源）处于同一产业链，但驱动逻辑完全不同：前者看纯碱价格和天然气成本，后者看IGBT供
  应和海外储能需求。把它们归为同一行业、限制合计仓位25%，会过度约束。

  修正：同行业 = 同一申万二级行业分类。
  产业链上下游不算同行业，但如果上下游的驱动因素高度相关（如锂矿+锂电池，都看锂价），需手动归入同一风险组。

  问题 15：整个策略缺少非ATR的简化版本

  ATR在大多数看盘软件上不直接显示，需要手动计算或使用公式编辑器。对于不用专业软件的投资者，策略的门槛太高。

  增加简化替代：
  如果不方便计算ATR，用以下近似：
  - ATR ≈ 近期20日平均振幅（最高-最低）的70%
  - 或直接用：20日平均真实波幅 ≈ 布林带宽度（上轨-下轨）/ 4
  - 再简：ATR ≈ 股价 × 20日历史波动率（需软件支持）

  如果以上都算不出来：
  - 用"近期最大单日跌幅 × 1.5"代替2倍ATR止损
  - 用"近期平均周振幅"代替ATR做仓位计算
  - 或者：放弃个股ATR，统一用股价的固定百分比（波动率适配交给仓位管理——高波动票降仓位）

  问题 16：没有定义"缩量"在盘中的实时可操作性

  ▎ 成交量处于近20日最低30%分位

  收盘后能判断，但盘中14:00做加仓决策时，当天的量还没走完。14:00的成交量只完成了全天的约70-80%。用这个不完整的量去和近20
  日的完整日量比，会系统性地低估。

  修正：用"14:00累计成交量 / 近20日同期平均累计成交量" < 0.7 作为缩量判断。
  或者更简单——等到14:50之后、接近收盘、全天量基本确定时再做判断。14:00的规则改为14:50。

  问题 17：最小建仓间隔用"交易日"——但中间隔了周末

  ▎ 距试探仓≥2个交易日

  周三建试探仓，周五可以加主仓(1)——中间隔了周四一天，满足2个交易日。但如果周三建仓后周四-周五连续大涨，两天的涨幅可能已
  经很大了。这是OK的——规则本来就没禁止这个。

  但反过来：周五建试探仓，下周二加主仓(1)——中间隔了周末两天，信息发酵了三天，周一如果大涨，周二加仓已经在高位。周末会放
  大信息不对称。

  修订：如果建仓间隔跨周末或节假日，最小间隔+1个交易日。 周五建的仓，下一笔至少等到下周三。

  ---
  五、复查上轮遗漏的交叉验证问题

  问题 18：主仓(1)缩量条件 + 价格条件同时满足的概率

  两个条件必须同时满足：① 回调到5日/10日均线 ② 缩量。在上升趋势中，回调到均线往往伴随放量（恐慌盘出逃）而非缩量。缩量回
  调到均线在强势股中非常罕见。两个条件同时满足的概率太低，会导致建仓进度拖很久。

  修正：OR 关系而非 AND。
  - 回调到均线（无论量）→ 可加
  - 或缩量企稳（无论是否碰均线）→ 可加
  - 两者同时满足是最优情况但不是必要条件

  修订版（第四轮）

  只列改动项，其余维持第三版。

  ---
  建仓

  试探仓比例：

  试探仓比例 = 根据波动率：
  - ATR/股价 > 4%（高波动）：3%
  - ATR/股价 2-4%（中等波动）：4%
  - ATR/股价 < 2%（低波动）：5%

  没有ATR时：近1个月单日最大跌幅>7%→3%，4-7%→4%，<4%→5%

  高开回避——改为累计条件：

  逻辑确立后：
  - 若当日涨幅≤4%：14:00后建试探仓
  - 若当日涨幅>4%：次日14:00再试
  - 从逻辑确立日算起，累计涨幅>10%且期间无单日跌幅>2%的回调：第4个交易日14:00直接建，不再等

  主仓(1)和主仓(2)——重新区分：

  ┌─────────┬──────────────────────────────────────────────────┬───────────────┐
  │  批次   │                  条件（OR关系）                  │     间隔      │
  ├─────────┼──────────────────────────────────────────────────┼───────────────┤
  │ 主仓(1) │ 首次回调至5日或10日均线；或缩量企稳（14:50判断） │ 距试探仓≥2日  │
  ├─────────┼──────────────────────────────────────────────────┼───────────────┤
  │ 主仓(2) │ 再次回落到10日或20日均线；或横盘≥5日未创新低     │ 距主仓(1)≥3日 │
  └─────────┴──────────────────────────────────────────────────┴───────────────┘

  如果主仓(1)买入后股价未再次回落而是直接上行——主仓(2)放弃，等确认仓。

  主仓B计划——明确豁免均线条件：

  第11个交易日仍未加主仓(1)：
  → 忽略均线回调条件，当日14:00后直接市价加主仓(1)
  → 主仓(2)放弃

  确认仓——增加涨停买不到终止：

  季报/月度数据验证后，2个交易日内14:00后加完。
  豁免高开限制。
  若2个交易日均一字涨停无法买入：放弃确认仓。

  跨周末/节假日的间隔调整：

  建仓间隔若跨周末或≥3天的节假日：最小间隔+1个交易日。
  例：周五建仓 → 下一笔不早于下周三。

  持仓

  移动止盈——改为阶梯表（放弃公式）：

  ┌─────────────────┬─────────────────┐
  │ 盈利（ATR倍数） │     止损位      │
  ├─────────────────┼─────────────────┤
  │ < 4             │ 成本价 - 2ATR   │
  ├─────────────────┼─────────────────┤
  │ 4 ~ 5           │ 成本价          │
  ├─────────────────┼─────────────────┤
  │ 5 ~ 6           │ 成本价 + 1.5ATR │
  ├─────────────────┼─────────────────┤
  │ 6 ~ 8           │ 当前价 - 3ATR   │
  ├─────────────────┼─────────────────┤
  │ 8 ~ 10          │ 当前价 - 4ATR   │
  ├─────────────────┼─────────────────┤
  │ > 10            │ 当前价 - 5ATR   │
  └─────────────────┴─────────────────┘

  使用时看盈利对应当前ATR的几倍，对号入座。阶梯跳变处（如5.9倍→6.0倍）按较宽松的执行（宁可晚收紧一步）。

  横盘处理——加大收紧力度：

  持仓超过预期时长的50%后，若股价仍在成本价±1.5ATR内：

  1. 硬止损收紧至成本价 - 1ATR
  2. 再加10个交易日观察期。10日后仍无突破（单日涨幅>3%或创近20日新高）→ 清仓
  3. 10日内突破 → 恢复正常ATR止损

  横盘期间行业指数已涨超10%而个股不动：
  → 减仓70%，不等时间止损

  相对强弱——明确算法：

  每日记录：个股当日涨跌幅% - 行业指数当日涨跌幅%（直接减，不累加）

  触发关注：个股5日累计涨跌幅 - 行业5日累计涨跌幅 < -5%
  （看盘软件上直接取5日涨幅两个数相减）

  触发减仓：个股10日累计涨跌幅 - 行业10日累计涨跌幅 < -8%
  → 减仓50%

  每周行业复查——修改不可操作项：

  □ 行业ETF本周资金净流入/流出（如有对应ETF，在天天基金等平台可查）
    - 替代原"北向资金行业统计"

  清仓

  跳空跌停——修正时间窗口：

  跳空跌停：
  - 9:15-9:20集合竞价挂跌停价卖出
  - 若未成交：维持挂单至收盘
  - 连续2个交易日未成交：
    第3天 9:15-9:19撤前日挂单 → 9:19-9:25重新挂跌停价
  - 连续3个交易日未成交且封单仍大：
    不再每日挂单。关注公司公告，等复牌或撬板。

  重新入场——加价格限制：

  重新入场条件（全部满足）：
  1. 距止损出场≥5个交易日
  2. 逻辑未被证伪
  3. 股价重新站稳20日均线
  4. 当前价 ≤ 原止损出场价 × 1.03（不得高于止损价3%以上）

  仓位上限减半。
  同一标的被止损2次后，该逻辑周期内不再碰。

  跳空向上减仓后——明确ATR基准：

  跳空>10%情绪驱动 → 14:00后减仓30%
  剩余70%的ATR止盈基准价 = 跳空前一日收盘价（不是跳空后的价格）

  多标的止损后的开仓限制——区分原因：

  个股原因导致的集体止损（各自独立触发不同因素）：
  → 一周内不开新仓

  大盘暴跌导致的集体止损（沪深300同日跌>3%）：
  → 大盘企稳后（沪深300连续2日收阳）即可开新仓，不等一周

  账户

  行业集中度——明确分类标准：

  同行业 = 同一申万二级行业（如"光伏设备"、"白酒"、"半导体"）
  同行业持仓 ≤ 2只
  同行业合计仓位 ≤ 25%

  以下情况需手动归入同一风险组（即使申万二级不同）：
  - 锂矿 + 锂电池（同看锂价）
  - 硅料 + 硅片（同看硅价）
  - 其他投入品价格高度相关的上下游

  判断标准：如果A公司的核心成本/收入因子与B公司高度重叠（相关系数肉眼可见），归入同组。

  ATR简化替代方案：

  无ATR数据时的近似：
  - ATR ≈ 近20日（最高-最低）均值的70%
  - 或：布林带(20)宽度 / 4

  如软件不支持以上计算，退化为固定百分比 + 仓位补偿：
  ┌──────────────┬──────────┬──────────┐
  │ 近1月最大单日跌幅 │ 止损距离  │ 单票上限  │
  ├──────────────┼──────────┼──────────┤
  │ > 8%          │ -10%     │ 8%       │
  │ 5-8%          │ -8%      │ 12%      │
  │ 3-5%          │ -6%      │ 15%      │
  │ < 3%          │ -5%      │ 18%      │
  └──────────────┴──────────┴──────────┘
  逻辑：高波动票用更宽的止损 + 更轻的仓位，风险金额大致相当。

  缩量判断——改为14:50执行：

  加仓日的缩量判断在14:50后进行（全天量已基本确定）。
  "缩量"= 当日成交量在近20个交易日中排名倒数6名以内（目测可判断）。
